execAsyncWithSuccess:AndFailure:

Excute the API request asynchronously with the given success and failure blocks.

- (void)execAsyncWithSuccess:(id)sucornil AndFailure:(id)failornil

Parameters

sucornil

The block to be executed if the request is successful. If it’s nil, the delegate’s [FHResponseDelegate requestDidSucceedWithResponse:] will be exectued.

failornil

The block to be executed if the request is failed. If it’s nil, the delegate’s [FHResponseDelegate requestDidFailWithResponse:] will be exectued.

Discussion

Excute the API request asynchronously with the given success and failure blocks.

This is the recommended way to execute the request.
